pipGuidanceDialog
service in module pipGuidance

Description

Reproduced API to show guidance dialog stretched out on a whole screen. It is included a navigation and allows users to go back through guide.

Methods

  • show(params, successCallback, cancelCallback)

    Shows guidance panel. Shown guidance can be close by click on backdrop space. Into callback function is passed nothing data.
